# BancaMobile Demo

Una página web sencilla de banca en línea que simula las operaciones básicas de un banco.

## Descripción

BancaMobile es una aplicación web demo que permite a los usuarios realizar operaciones bancarias básicas de forma segura y sencilla. Este proyecto está diseñado con fines educativos y de demostración.

## Características

- **Inicio de sesión seguro** - Autenticación de usuarios
- **Consulta de saldo** - Visualización del balance de cuenta
- **Transferencias** - Envío de dinero entre cuentas
- **Historial de movimientos** - Registro de transacciones realizadas
- **Perfil de usuario** - Gestión de datos personales

## Tecnologías

- HTML5
- CSS3
- JavaScript

## Pre-requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Conexión a internet (para cargar recursos externos)

## Pre-requisitos de desarrollo

- Editor de código (Visual Studio Code, Sublime Text, etc.)
- Git (opcional, para clonar el repositorio)

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/alemcuevas/bancamobile_demo.git
   ```

2. Navega al directorio del proyecto:
   ```bash
   cd bancamobile_demo
   ```

3. Abre el archivo `index.html` en tu navegador o usa un servidor local.
